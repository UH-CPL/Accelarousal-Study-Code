# Driver Prediction

## Getting Started
### Prerequisites
- R and RStudio
- Required packages

### Installing R Packages
To install required R packages, run script `setup.R` in each study folder.

## Folder Structure
- `ord`: On-road Driving Study.
- `ttd1`: Test Track Driving 1.
- `utils`: Common utility scripts.
